# Задачи по программированию на C++

## Условия задач

### Задача 1. Температура
В массиве записаны данные температуры в течение года. Вывести среднюю температуру за каждый месяц. Вывести самый холодный и самый теплый месяцы.

**Ограничения:**
- Использовать только массивы и класс string (без коллекций)
- Допускается использование собственных функций
- Запрещены коллекции и иные структуры данных

### Задача 2. Удаление слов
Дана строка. Удалить слова со M по N.

**Ограничения:**
- Использовать только массивы и класс string
- Допускается использовать обработку исключительных ситуаций (try-catch)
- Допускается использование собственных функций
- Запрещены коллекции и иные структуры
